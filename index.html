CryptoPips Real-Time Cryptocurrency Analytics

A responsive cryptocurrency dashboard was developed using React, Tailwind CSS, Chart.js, Alpha Vantage, and CoinGecko APIs to deliver real-time price tracking, historical data visualization, and market insights. Featuring interactive charts and live updates, the platform ensures an optimized user experience for analyzing cryptocurrency trends.

Qr-code Gen

This tool instantly generates QR codes for URLs and messages, with optional real-time encryption for added privacy. Built for speed and simplicity, it uses an API call to ensure seamless performance. A built-in history feature helps recover previously encoded data, so users can quickly retrieve past URLs or messages in case of data loss.

Wordwisp

This webpage is a dynamic language detection and translation tool designed to enhance user accessibility and communication. It automatically identifies the language of input text and seamlessly translates it based on the user’s selected language preference. Developed using the Chrome Developer Tools suite, the project leverages modern web technologies to ensure performance, responsiveness, and intuitive user interaction.
